Glyph

Each utterance can be annotated by statistical measures that are calculated by the framework. Shallow text mining and linguistic processing yields a set of 53 features that encode various properties of the debate. All of them serve as operationalizing features for analyzing communicative strategies in deliberative communication.

Glyph Encoding:
The glyph is limited up to 7 measures that can be chosen from the list. The List contains only measures which have a variance greater zero to reduce the amount of measures, they were also sorted in descending order of standard deviation. The larger the standard deviation the more interesting, because there is more change in the data.

Each measure is represented as a circle and encodes the value in the form of color. Around this circle the min and max value of the respective measure is mapped, which not only indicates the range but also represents the data type. The left side of the circle border encodes the min value and the right side the max value.
