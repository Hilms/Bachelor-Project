Topics

Topic modeling provides a suite of algorithms to discover a set of hidden thematic structures ("topics" - recurring themes that are discussed) in large collections of texts. Topic modeling refers to the task of identifying topics that best describes a set of documents and will only emerge during the topic modelling process (therefore called latent). The results of topic modeling algorithms can be used to summarize, visualize, explore, and theorize about a corpus.

Topic modeling is a form of unsupervised statistical machine learning - only the words in the document are modeled. It is like document clustering but with with one important advantage - instead of each document belonging to a single cluster or topic, a document can belong to many different clusters or topics.

Latent Dirichlet Allocation (LDA) is an unsupervised algorithm and determines a fixed set of topics. Each topic is represented/described by a set of words. The goal of LDA is to map all the documents to the topics in a way, such that the words in each document are mostly captured by those imaginary topics.

Besides LDA another one is provided, the Incremental Hierarchical Topic Model (IHTM). IHTM supports the active incorporation of the user’s domain knowledge in every step through explicit model manipulation interactions. In addition, users can initialize the model with expected topic seeds. This supervised human-in-the-loop process targets maximum improvement for the topic modelling.
